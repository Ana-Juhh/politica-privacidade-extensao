Política de Privacidade

Esta extensão coleta dados de localização (latitude, longitude e precisão)
exclusivamente para fins institucionais de rastreamento de dispositivos
Chromebook pertencentes à organização.

Os dados coletados são utilizados apenas para controle e segurança dos
dispositivos e não são compartilhados com terceiros, utilizados para fins
comerciais ou publicidade.

As informações são transmitidas de forma segura para um servidor institucional
e armazenadas conforme políticas internas da organização.

Nenhuma informação pessoal sensível é coletada além do necessário para o
funcionamento da extensão.
